Schedule Manipulation Analysis Tool

Paste a project schedule, click Analyze, and see its critical path, finish date, and the 14 DCMA quality checks.

1. Your schedule (JSON)

Open .json loads the tool's own format. Import reads an MS Project XML export, a Primavera P6 .xer, or a native MS Project .mpp (best-effort — eyeball the result). .mpp needs Java + the optional MPXJ library (the launcher installs it); if it's unavailable, use File → Save As → XML in MS Project and import the .xml. Time is in working minutes (1 work day = 8h = 480); "FS" = task B starts after task A finishes.

Click Analyze schedule to see the report.
